TABLA MOVIE¶

Esta tabla nos muestra los datos caracersticos de una pelcula (movie): Titulo, fecha de creacin o lanzamiento (movie_date), formato (movie_format), un id que se relaciona con la tabla genero (movie_genre_id), usuario (created_by_user), fecha(created_date), fecha acualizacion(updated_date).

TABLA GENRE¶

Contiene datos de los generos o catalogacin de cada pelicula, pudiendo ser esta de tipo accion, comedia, romance, etc., los datos principales que se guardan son: id (genre_id), nombre de genero(genre_name), usuario quien creo o ingreso el dato(created_date), fecha de ingreso del dato(updated_date), fecha de actualizacion(update_date).

TABLA PERSON¶

Esta tabla contiene los datos de la personas que tienen participacin dentro una pelicula, entre ellas actores, directores, guionistas etc. Los datos que se guardan de las personas son: Id (person_id), Nombre de la persona, (person_name), pais (person_country), fecha_de nacimiento (person_dob), un id de padre o madre de otra dato de persona(person_parent_id), datos de creacion o actualizacin del dato(created_by_user),(created_date),(updated_date)

TABLA ROLE¶

En esta tabla se encuentran registrados los roles que cumple cada persona dentro la pelicula, esto pueden ser acores, musicos, guionitas, etc. En esta tabla se registra un id (role_id), role_name, datos de creacion o actualizacin del dato.

TABLA MOVIE_PERSON¶

Esta tabla representa la relacin entre pelcula, persona y rol. Ademas se registra si la persona ha recibido alguna premiacin. Al igual que las otras tablas contiene datos del registro o modificacin del dato.
